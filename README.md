# DIZI Ordering Agent Skill

AI-агент для оформления B2B-заказов через платформу [DIZI](https://dizi.procorp.asia) —
сервис заказов магазинов у поставщиков (FMCG, стройматериалы, фарма и др.)
на рынках Кыргызстана, Казахстана и Узбекистана.

## Что умеет агент

- 🔍 Ищет товары по каталогам поставщиков (нечёткий поиск по названию/бренду)
- 📦 Учитывает **квант поставки** и предупреждает о корректировке количества
- 🛒 Формирует корзину, группирует по поставщикам
- 🌿 Даёт **сезонные рекомендации** с учётом даты и рынка ЦА
- 📊 Применяет **ABC-анализ** по истории заказов магазина
- ✅ Оформляет заказ только после явного подтверждения пользователя

## Структура

```
dizi-skill/
├── SKILL.md                    # Основной скилл — инструкции для агента
├── system-prompt.md            # Готовый system prompt для вставки в ИИ
└── references/
    ├── tools.json              # JSON-схемы инструментов (function calling)
    └── seasonal-calendar.md   # Сезонный календарь ЦА + праздники
```

## Быстрый старт

### Вариант 1 — System Prompt (ChatGPT, Gemini, Claude, голосовой ассистент)

1. Открой файл `system-prompt.md`
2. Скопируй содержимое блока ``` ... ```
3. Вставь как системный промпт в своего ИИ-ассистента — менять ничего не нужно
4. При старте агент сам попросит пользователя ввести API-ключ и проверит его

### Вариант 2 — Function Calling / Tool Use (для разработчиков)

1. Подключи `references/tools.json` как определения инструментов
2. Настрой HTTP-клиент с заголовком `Authorization: Bearer {USER_API_KEY}`
3. Base URL: `https://dizi.procorp.asia/api`
4. Используй `SKILL.md` как системный промпт агента

### Вариант 3 — Claude Skill (для Claude.ai)

Загрузи `SKILL.md` как скилл в Claude через интерфейс управления скиллами.

## Получение API-ключа

1. Зарегистрируйся на [dizi.procorp.asia](https://dizi.procorp.asia)
2. Перейди в **Профиль → Настройки**
3. Скопируй поле **API Key**

- Минимум одна активная торговая точка
- API-ключ из профиля
